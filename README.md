CALayer-DPUtils
===============

## Documentation

### CALayer (FrameUtils)

You can show the status of indeterminate tasks using:

```objective-c
- (CGFloat) width;
- (CGFloat) height;
- (CGFloat) top;
- (CGFloat) left;
- (CGSize) size;

- (void) setWidth: (CGFloat) width;
- (void) setHeight: (CGFloat) height;
- (void) setTop: (CGFloat) top;
- (void) setLeft: (CGFloat) left;

- (void) setSize: (CGSize) size;
```
